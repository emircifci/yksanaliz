# YKS Asistanı - GitHub Pages

Bu klasör, YKS Asistanı uygulaması için GitHub Pages web sitesini içerir.

## Dosyalar

- `index.html` - Anasayfa
- `privacy-policy.html` - Gizlilik Politikası (KVKK uyumlu)
- `terms-of-service.html` - Kullanım Şartları
- `support.html` - Destek sayfası

## GitHub Pages Kurulumu

1. Bu `docs` klasörünü GitHub repository'nize yükleyin
2. GitHub repository ayarlarına gidin
3. "Pages" bölümüne gidin
4. Source olarak "Deploy from a branch" seçin
5. Branch olarak "main" (veya "master") seçin
6. Folder olarak "/docs" seçin
7. Save butonuna tıklayın

## URL'ler

GitHub Pages aktif olduktan sonra, web siteniz şu URL'de olacak:
- `https://[kullanici-adi].github.io/[repo-adi]/`

Örnek:
- `https://yksanaliz.github.io/yksanaliz/`

## App Store Connect URL'leri

GitHub Pages aktif olduktan sonra, App Store Connect'e şu URL'leri ekleyin:

- **Privacy Policy URL:** `https://[kullanici-adi].github.io/[repo-adi]/privacy-policy.html`
- **Terms of Service URL:** `https://[kullanici-adi].github.io/[repo-adi]/terms-of-service.html`
- **Support URL:** `https://[kullanici-adi].github.io/[repo-adi]/support.html`
- **Marketing URL (Opsiyonel):** `https://[kullanici-adi].github.io/[repo-adi]/`
